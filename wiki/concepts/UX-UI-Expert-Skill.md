# Concept: UX/UI Expert Skill

## Overview
A specialized agent skill designed to provide expert-level guidance on User Experience (UX) and User Interface (UI) design principles.

## Core Sources
- **Vercel Web Interface Guidelines**: Actionable technical rules for high-quality web UIs (keyboard accessibility, focus management, optimistic updates, etc.).
- **Laws of UX**: Psychological principles including Hick's Law, Fitts's Law, and Jakob's Law.
- **Modern Web Standards**: Emphasis on GPU-accelerated animations, responsive design, and semantic HTML.

## Key Capabilities
- **Design Reviews**: Auditing existing code for accessibility and visual consistency.
- **Prototyping**: Generating modern, intuitive layouts using CSS Grid/Flexbox.
- **Best Practices**: Guiding the implementation of micro-interactions and high-performance animations.

## Usage
Triggered by keywords like "review UI", "improve design", or "design a new page".

## Implementation Details
Installed as a Gemini CLI skill at the user level: `ux-ui-expert`.
